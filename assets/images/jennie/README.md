# Jennie 参考图片文件夹

## 使用说明

请将您收集的 Jennie 图片放置到这个文件夹中,命名为:

- `jennie_1.jpg`
- `jennie_2.jpg`
- `jennie_3.jpg`
- `jennie_4.jpg`
- `jennie_5.jpg`
- `jennie_6.jpg`
- `jennie_7.jpg`
- `jennie_8.jpg`
- `jennie_9.jpg`
- `jennie_10.jpg`

可以继续添加更多图片(jennie_11.jpg, jennie_12.jpg等),只需在代码中相应添加即可。

## 图片要求

- 格式: JPG或PNG
- 建议尺寸: 不要太大,推荐1000x1000以下
- 内容: Jennie的各种风格照片 - 机场街拍、时尚大片、舞台照、生活照等

## 工作原理

当用户选择"Jennie同款"模板时:
1. 系统会随机选择1-2张参考图片
2. AI会学习这些图片中的:
   - 表情神态
   - 妆容发型
   - 姿态动作
   - 服装配饰
   - 背景场景
   - 摄影风格
3. 保留用户的脸部特征和身材
4. 生成Jennie风格的照片,无水印
