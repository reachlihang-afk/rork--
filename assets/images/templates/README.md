# 模板预览图片文件夹

## 使用说明

这个文件夹存放首页模板卡片的预览图片。

## 需要的图片

请将以下图片放置到此文件夹:

1. **jennie.jpg** - Jennie模板预览图
2. **princess.jpg** - 公主模板预览图  
3. **cyberpunk.jpg** - 赛博朋克模板预览图
4. **sports.jpg** - 运动装模板预览图
5. **vintage.jpg** - 复古风模板预览图
6. **hero-preview.jpg** - 首页Hero卡片右上角的预览图

## 图片要求

- **格式**: JPG 或 PNG
- **尺寸**: 建议 400x533 像素 (3:4比例)
- **大小**: 每张不超过 500KB
- **内容**: 展示对应模板的效果预览

## 备用方案

如果暂时没有图片,可以:
1. 使用占位图片(纯色背景 + 图标)
2. 使用网络图片URL(修改代码中的 require 为 { uri: 'URL' })
3. 暂时注释掉图片显示代码
