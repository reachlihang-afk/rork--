### 3. 用户头像/爱心图标 ✅✅✅

**重要更新**：一键换装卡片右侧的图标现在会根据登录状态变化！

#### 未登录状态
```
显示: ❤️ (红色爱心emoji)
背景: 深色圆形
```

#### 已登录状态
```
显示: 用户头像
样式: 圆形头像，白色边框
大小: 80x80
```

**实现逻辑**:
```typescript
{isLoggedIn && user?.avatar ? (
  <Image source={{ uri: user.avatar }} style={styles.heroAvatar} />
) : (
  <Text>❤️</Text>
)}
```
