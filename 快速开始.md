# 快速开始指南

## 🚀 Windows 用户快速启动

### 最简单的方式
双击运行 `start-with-utf8.bat` 文件即可启动项目！

这个脚本会：
- ✅ 自动设置 UTF-8 编码（避免中文乱码）
- ✅ 检查并安装依赖
- ✅ 启动 Expo 开发服务器

### 启动后的操作
开发服务器启动后，你可以：
- 按 **a** 键 → 在 Android 模拟器中打开
- 按 **i** 键 → 在 iOS 模拟器中打开（仅 Mac）
- 按 **w** 键 → 在浏览器中打开
- 扫描二维码 → 在手机上打开（需要安装 Expo Go）

## 🔧 如果遇到编码问题

如果看到中文显示为乱码，运行以下命令修复：

### 方式 1：使用修复脚本
```powershell
powershell -ExecutionPolicy Bypass -File fix-encoding.ps1
```

### 方式 2：手动设置
在 PowerShell 中运行：
```powershell
chcp 65001
```

详细的编码问题解决方案请查看 `编码问题解决方案.md`

## ❓ 常见问题

### Q: 启动失败怎么办？
A: 
1. 确保已安装 Node.js 和 Bun
2. 删除 `node_modules` 文件夹
3. 重新运行 `bun install`
4. 再次启动项目

### Q: 中文显示乱码？
A: 使用 `start-with-utf8.bat` 启动，或运行 `fix-encoding.ps1` 修复

### Q: 如何在手机上测试？
A: 
1. 手机和电脑连接同一 WiFi
2. 手机安装 Expo Go（iOS/Android 应用商店）
3. 扫描终端显示的二维码

### Q: 模拟器连不上？
A: 运行 `adb reverse tcp:8081 tcp:8081`
