## 📱 使用流程

### 用户视角

#### 1. Template模式
```
上传照片 → 选择模板 → 设置选项 → 生成
```

#### 2. Custom模式
```
上传照片 → 上传参考图 → 设置选项 → 生成
```

#### 3. Pro Style模式
```
上传照片 → 点击达人卡片 → 进入Look集合 → 选择Look → 返回 → 生成
```

### 技术流程

#### Pro Style数据流
```
influencer-collection/[id].tsx (选择Look)
    ↓ router.push with params
outfit-change-new.tsx (接收lookPrompt)
    ↓ useEffect
setState (显示已选择Look)
    ↓ 点击生成
buildPrompt (使用lookPrompt)
    ↓
AI生成
```
