# 网图克星 (PicSeek) - 产品文档

## 📱 产品概述

**网图克星** 是一款基于 AI 技术的多功能图片处理应用，集成了照片验证、图片溯源和 AI 换装等核心功能。通过先进的人工智能算法，为用户提供图片真实性检测、图片来源追踪和创意换装服务。

### 产品定位
- **目标用户**：关注照片真实性、需要图片溯源或创意换装的用户
- **核心价值**：AI 驱动的图片真伪鉴别、智能图片溯源、趣味换装体验
- **应用场景**：社交验证、内容审核、娱乐创作、电商防骗等

### 技术栈
- **前端框架**：React Native + Expo
- **路由管理**：Expo Router
- **状态管理**：Zustand + Context API
- **国际化**：i18next + react-i18next
- **UI 组件**：Lucide React Native Icons
- **AI 服务**：Rork AI Toolkit SDK
- **平台支持**：iOS、Android、Web

## 🎯 核心功能

### 1. 照片验证 (Photo Verification)

#### 功能描述
通过 AI 算法分析照片的真实性，检测是否经过 PS 修图处理，为用户提供可信度评分和详细分析报告。

#### 主要特性
- **自拍照对比验证**
  - 支持上传 1-5 张自拍照作为参照
  - 系统自动保存已上传的自拍照
  - 支持删除和重新上传自拍照

- **智能分析维度**
  - 面部相似度 (Facial Similarity)
  - 皮肤纹理 (Skin Texture)
  - 身体比例 (Proportions)
  - 光影效果 (Lighting)

- **验证结果评级**
  - 🟢 **真实 (Authentic)**：90-100 分，照片未经修改
  - 🟡 **轻微编辑 (Slightly Edited)**：70-89 分，有轻微美颜
  - 🟠 **重度编辑 (Heavily Edited)**：50-69 分，经过大幅修改
  - 🔴 **可疑 (Suspicious)**：0-49 分，严重失真或 AI 生成

- **验证报告内容**
  - 可信度评分（0-100分）
  - 主体类型识别（人物/动物/建筑/物品等）
  - 详细分析说明
  - 唯一验证码
  - 设备 ID 记录

- **高级功能**
  - 验证结果分享到社交广场
  - 验证报告截图保存到相册
  - 自定义验证说明
  - 历史验证记录查询

#### 使用流程
1. 首次使用需上传 1-5 张自拍照作为参考
2. 选择待验证的照片（拍照或从相册选择）
3. AI 自动分析并生成验证报告
4. 查看详细分析结果和评分
5. 可选：分享到社交广场或保存报告

#### 计费规则
- 每次验证消耗 **100 金币**
- 验证失败不扣除金币

### 2. 网图溯源 (Image Source Tracking)

#### 功能描述
通过 AI 技术识别图片内容，帮助用户找出图片的可能来源、相关信息和背景知识。

#### 主要特性
- **智能内容识别**
  - 自动识别图片中的人物、动物、植物、建筑等
  - 提取关键词和描述信息
  - 识别知名人物、地标等实体信息

- **实体信息展示**
  - 实体名称
  - 详细介绍
  - 实体类型（人物/动物/植物/其他）

- **溯源建议**
  - 可能的图片来源列表
  - 相关搜索关键词
  - 反向搜索建议

- **历史记录管理**
  - 自动保存溯源历史
  - 支持查看历史溯源结果
  - 可删除单条记录或清空全部

#### 使用流程
1. 点击"找出处"功能入口
2. 上传或拍摄要溯源的图片
3. AI 自动分析图片内容
4. 查看识别结果和溯源建议
5. 根据关键词进行进一步搜索

#### 计费规则
- 每次溯源消耗 **50 金币**
- 识别失败不扣除金币

### 常见问题

#### Q1: 为什么需要上传自拍照？
A: 自拍照用于照片验证功能，作为对比参照，帮助系统更准确地判断待验证照片是否被修改。

#### Q2: 金币不够怎么办？
A: 点击个人中心的"金币充值"按钮，选择合适的套餐进行充值。

#### Q3: 换装效果不满意可以重新生成吗？
A: 可以，每次生成都会消耗金币，建议选择合适的照片和模板以获得最佳效果。

#### Q4: 历史记录会保存多久？
A: 历史记录会永久保存在本地，除非您手动删除。

#### Q5: 支持哪些图片格式？
A: 支持 JPG、PNG 等常见图片格式。

#### Q6: 可以切换语言吗？
A: 可以，在个人中心点击"语言设置"即可切换。
