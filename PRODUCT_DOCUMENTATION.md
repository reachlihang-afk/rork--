# 百变星君 - 产品文档（主打：一键换装）

## 🪄 产品概览
- **名称**：百变星君（原“网图克星”已更名）
- **核心卖点**：一键换装，秒变百种造型
- **目标人群**：年轻女性/时尚爱好者/社交分享用户
- **平台**：iOS / Android / Web

---

## 🎯 主打功能：一键换装

### 1) 模板换装（预设样式，点选即用）
- 入口：首页「一键换装」 → 选择模板 → 开始生成 → 弹窗提示「图片已生成」
- 操作：上传/拍照 → 选模板 → 生成 → 下载/发布/放大/长按保存
- 随机装/公主装/普通模板均可在大图长按保存到相册，缩略图点击放大。

**模板列表（持续扩充，已支持 30+）**
- 🎲 随机装：100+风格池，每次完全不同（含配饰/包/鞋的完整穿搭）
- 👸 童话公主装：10种迪士尼公主随机（灰姑娘/白雪公主/贝儿/爱丽儿/奥罗拉/茉莉/长发公主/艾莎/莫阿娜/蒂安娜）

**模板内随机多样化**
- 每个模板都会在本类别内随机变换剪裁、颜色、图案、面料、配饰，避免重复；正装/比基尼/运动装等都会生成不同款式与配色。

### 2) 一键穿搭（自定义单品搭配）
- 上传最多2张服饰/配饰图片，AI 自动贴合可见身体部位
- 规则：不补全缺失肢体；只作用可见区域；保持人脸/表情/发型不变

### 3) 体验与交互
- 生成完成弹窗提示：「图片已生成」
- 原图/结果图对比；点击放大，长按保存
- 支持发布到广场，带模板彩色徽章
- 模板卡片可「展开更多」，今日免费次数醒目显示

### 4) 计费/免费
- 未登录：每日1次免费
- 已登录：每日5次免费
- 超出：200金币/次；注册赠送1000金币

## 🌟 次要功能（保留）
- **照片验证（人照合一）**：真伪评分与结论，可分享/历史查看
- **图片找出处**：识别图片实体（仅下载结果，不再分享至广场）
- **广场**：查看/点赞/评论换装与验证帖子，图片可放大与长按保存
- **历史记录**：换装/验证/找出处三类；换装记录详情支持放大、长按保存、分享、删除；换装记录可设隐私可见性
- **个人中心**：金币余额、每日免费次数（换装/验证/找出处）、隐私设置、语言切换（中/英/日/韩）

## ⚙️ 技术与架构（简版）
- RN + Expo Router；状态：React Context
- 图片：expo-image-picker / image-manipulator；Base64 兼容保存
- AI：`https://toolkit.rork.com/images/edit/`（换装） / `verify/` / `analyze/`
- 存储：AsyncStorage（按用户隔离）；换装历史最多5条自动轮换
- 国际化：i18next（中/英/日/韩）
