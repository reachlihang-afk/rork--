# 百变星君 - 产品文档

## 🪄 产品概览
- **产品名称**：百变星君
- **核心定位**：AI 驱动的一键换装社交平台
- **核心卖点**：秒变百种造型，轻松玩转时尚
- **目标用户**：年轻女性 / 时尚爱好者 / 社交分享达人
- **支持平台**：iOS / Android / Web

---

## ✨ 核心功能：一键换装

### 功能介绍
百变星君提供强大的 AI 换装功能，用户只需上传照片，即可一键生成各种风格的时尚造型。无需试穿，无需购买，就能看到自己穿上不同服装的效果。

### 两种换装模式

#### 1. 模板换装（预设风格，一键生成）

**操作流程**：
1. 上传照片或拍照
2. 选择喜欢的模板风格
3. 点击生成
4. 获得精美换装结果

**模板特色**：
- ✅ **30+ 种模板**：从日常到派对，从优雅到前卫
- ✅ **智能多样化**：同一模板每次生成不同款式和配色

#### 2. 一键穿搭（自定义搭配）

**操作流程**：
1. 上传照片或拍照
2. 上传1-2张服饰/配饰参考图
3. AI 自动将参考服饰融合到照片中
4. 获得个性化穿搭效果

**特点**：
- ✅ 支持最多2张参考图片
- ✅ AI 智能贴合身体可见部位
- ✅ 自然融合，保持原始姿态

### 技术优势

**AI 智能处理**：
- 🧠 保持面部特征、表情、发型不变
- 🎯 只改变服装，不修改身体姿态
- 📐 精确识别可见部位，不会补全缺失部分
- 🎨 自然融合，无违和感

**生成效果**：
- 高清输出，细节丰富
- 自然光影，真实质感
- 完整配饰搭配
- 风格多样不重复

### 使用体验

**便捷操作**：
- 📸 支持拍照或相册选择
- ✨ 可选美颜功能优化照片
- 🎭 点击放大查看细节
- 💾 长按保存到相册

**生成提示**：
- 生成完成后弹窗提示
- 显示生成进度和用时
- 失败自动重试提示

**互动分享**：
- 发布到广场与好友分享
- 点赞、评论互动交流
- 模板徽章清晰展示

## 📱 其他功能

### 广场（社交互动）
- **浏览作品**：查看其他用户的换装作品
- **点赞评论**：为喜欢的作品点赞和留言

- **换装历史**：保存所有换装记录，方便回顾
- **详情查看**：查看原图、结果图、使用模板
- **快速操作**：
  - 📥 下载保存到相册
  - 🔄 重新生成（使用相同模板）
  - 📤 发布到广场分享

- **隐私设置**：可设置记录是否允许分享到广场

### 个人中心
- **账号管理**：
  - 登录/注册功能
  - 个人资料编辑（昵称、头像）
- **金币系统**：
  - 查看金币余额
  - 充值购买金币
  - 查看消费记录
- **免费额度**：
  - 未登录：每日 1 次免费换装
  - 已登录：每日 5 次免费换装
- **系统设置**：
  - 多语言切换（中文/英文/日文/韩文）
  - 隐私设置
  - 好友管理

## 🎯 用户使用流程

### 新用户初体验
1. 打开 APP → 进入首页
2. 点击「一键换装」
3. 上传/拍摄照片
4. 选择喜欢的模板（如"Jennie同款"）
5. 点击「开始生成」
6. 等待 10-30 秒
7. 弹窗提示「图片已生成」
8. 查看结果 / 保存 / 分享到广场

### 老用户日常使用
1. 进入 APP → 首页显示最近换装记录
2. 快速开始新换装 or 查看历史记录
3. 在广场浏览其他人的作品
4. 为喜欢的作品点赞评论
5. 管理个人主页和金币余额

## 🛠️ 技术架构（简述）

### 前端技术
- **框架**：React Native + Expo
- **路由**：Expo Router
- **状态管理**：React Context API
- **UI组件**：自定义组件库
- **图标**：Lucide React Native

### 后端服务
- **AI引擎**：`https://toolkit.rork.com/images/edit/`
- **图片处理**：Base64 编码/解码
- **存储**：AsyncStorage（本地持久化）

### 多语言
- **i18n框架**：react-i18next
- **支持语言**：中文、英文、日文、韩文
