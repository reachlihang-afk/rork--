## 📞 遇到问题?

### 1. 页面打不开
- 检查文件路径是否正确
- 检查是否有语法错误
- 运行 `npm run lint`

### 2. 生成失败
- 查看console.log输出
- 检查网络连接
- 检查API地址: https://toolkit.rork.com/images/edit/

### 3. Influencer页面打不开
- 确认 `influencer-collection` 文件夹已创建
- 确认 `[id].tsx` 文件存在
